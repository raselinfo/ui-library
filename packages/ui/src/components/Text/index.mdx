# Text

Use the Text component to render text. It supports all HTML5 text elements. It also supports the following custom elements: `span`, `strong`, `em`, `small`, `mark`, `del`, `ins`, `sub`, `sup`, `code`, `kbd`, `samp`, `var`, `abbr`, `address`, `cite`, `bdo`, `bdi`, `time`, `dfn`, `q`, `blockquote`, `pre`, `ul`, `ol`, `li`, `dl`, `dt`, `dd`, `div`, `p`, `a`, `h1`, `h2`, `h3`, `h4`, `h5`, `h6`.
